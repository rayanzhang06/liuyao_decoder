# 盲派文献库

本目录存放盲派的经典文献和口诀。

## 文献格式规范

每篇文献应遵循以下Markdown格式：

```markdown
# 《书名或口诀集》

## 篇章名

### 口诀或说明

口诀正文内容...
```

## 文献列表

- `mangpai_koujue.txt` - 盲派口诀（占位符，待添加实际内容）

## 添加文献

1. 将文献文本转换为.txt格式
2. 按照上述格式规范整理
3. 放入本目录
4. 重启应用或运行索引重建命令
